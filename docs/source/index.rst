Welcome to lungsim-post's documentation!
========================================

Hello everyone! Welcome to lungsim-post!

This codebase is a python library which can be used to carry out common post-processing tasks on models generated by the lungsim software, this includes analysing model results on a per-voxel basis (akin to imaging), analysing particular pathways in a tree structure, and converting file types.
